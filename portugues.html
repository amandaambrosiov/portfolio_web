Amanda Ambrosio

Desenvolvedora web & Designer Grafico

Olá, seja bem-vindo! Sou Amanda, uma jovem desenvolvedora front-end apaixonada por criar experiências digitais incríveis. Meu objetivo é trazer à vida websites e aplicativos que se destacam pela funcionalidade e design atraente. Apesar de minha idade, estou determinada a me aprofundar no mundo da programação e design. Estou constantemente aprendendo e buscando maneiras de aprimorar minhas habilidades, sempre em busca de projetos empolgantes para trabalhar. Este é o meu espaço para compartilhar minha jornada de desenvolvimento front-end e mostrar os projetos em que tenho trabalhado. Convido você a explorar meu portfólio e acompanhar meu crescimento à medida que continuo a me aperfeiçoar nessa emocionante jornada de criação digital.

Habilidades

Git

Github

Bootstrap

Javascript

O Git me ajudou de diversas maneiras na minha jornada como desenvolvedora. Ele simplificou o controle de versão do código, permitindo que eu acompanhasse e revertesse alterações com facilidade. Além disso, a colaboração se tornou mais eficiente, tornando o trabalho em equipe mais produtivo.

O GitHub tem sido fundamental na minha experiência como desenvolvedora. Ele fornece um espaço centralizado para armazenar e compartilhar meus projetos, facilitando a colaboração com outros desenvolvedores. Além disso, a visibilidade que o GitHub oferece ao meu trabalho é valiosa para construir minha reputação na comunidade de desenvolvimento.

Bootstrap tem sido um aliado essencial na minha jornada como desenvolvedora. Ele oferece uma ampla gama de componentes e estilos prontos para uso, acelerando o processo de criação de interfaces atraentes e responsivas. Sua estrutura CSS facilita a personalização e manutenção de projetos web. Além disso, a compatibilidade multiplataforma do Bootstrap garante que minhas aplicações funcionem bem em diferentes dispositivos, economizando tempo e esforço.

JavaScript tem sido fundamental na minha experiência como desenvolvedora. Ele é a linguagem de programação essencial para o desenvolvimento web, permitindo interatividade e dinamismo em sites e aplicativos. Com JavaScript, posso criar recursos interativos, validar formulários e melhorar a experiência do usuário.
